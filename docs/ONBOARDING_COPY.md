# Onboarding Copy

이 문서는 앱 화면, 안내 메시지, 버튼, 빈 상태에 사용할 수 있는 짧은 한국어 문구 모음이다.

## 1. Product One-Liner

카톡에 흘러가는 기도제목을 함께 기억하는 기록으로.

## 2. First Screen Copy

Title:

소그룹 체크인을 시작해요

Body:

이메일로 가입하거나 로그인한 뒤, 초대코드로 동행방에 들어갈 수 있어요.

Alternative body for leaders:

리더라면 새 동행방을 만들고, 멤버에게 초대코드를 공유해 주세요.

## 3. Auth Copy

Login tab:

로그인

Sign-up tab:

새 계정

Login helper:

이미 만든 계정이 있다면 이메일과 비밀번호로 들어갈 수 있어요.

Sign-up helper:

처음이라면 가입 확인 메일을 받을 수 있는 이메일로 계정을 만들어요.

Email validation:

이메일 형식을 확인해 주세요.

Password validation:

비밀번호를 6자 이상 입력해 주세요.

Check-email notice:

가입 확인 메일을 보냈어요. 메일함에서 확인을 완료한 뒤 다시 로그인해 주세요.
