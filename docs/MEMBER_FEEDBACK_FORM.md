# Member Feedback Form

이 문서는 Google Forms, Typeform, 카카오톡 설문, 또는 수기 인터뷰에 그대로 옮겨 사용할 수 있다.

## Form Title

동행방 베타 멤버 피드백

## Intro

동행방을 사용해 주셔서 감사합니다.  
이 설문은 소그룹 멤버가 부담 없이 체크인하고 기도제목을 남길 수 있었는지 확인하기 위한 것입니다.  
개인적인 기도제목 내용은 적지 않아도 됩니다.

## Section 1. Basic Info

1. 이름 또는 별칭  
   - Short answer

2. 동행방을 어떤 기기로 사용했나요?  
   - Multiple choice
   - 아이폰
   - 안드로이드폰
   - 태블릿
   - 컴퓨터

3. 이번 주에 동행방을 몇 번 열어봤나요?  
   - Multiple choice
   - 1번
   - 2-3번
   - 4번 이상
   - 기억나지 않음

## Section 2. First Use

4. 가입하고 초대코드로 들어가는 과정은 쉬웠나요?  
   - Linear scale 1-5
   - 1: 매우 어려웠다
   - 5: 매우 쉬웠다

5. 첫 화면에서 무엇을 해야 하는지 이해하기 쉬웠나요?  
   - Linear scale 1-5

6. 막힌 부분이 있었다면 어디였나요?  
   - Paragraph

## Section 3. Check-In

7. 오늘 체크인을 20초 안에 남길 수 있었나요?  
   - Multiple choice
   - 예
   - 거의 가능했다
   - 아니오

8. 체크인 문구가 부담스럽지 않았나요?  
   - Linear scale 1-5

9. 체크인이 출석 확인이나 평가처럼 느껴졌나요?  
   - Multiple choice
   - 전혀 그렇지 않았다
   - 조금 그랬다
   - 많이 그랬다

10. 체크인에서 바꾸고 싶은 점이 있다면 무엇인가요?  
    - Paragraph

## Section 4. Prayer Requests

11. 기도제목을 카드로 남기는 방식이 이해되었나요?  
    - Linear scale 1-5

12. 공개 범위가 충분히 분명했나요?  
    - Linear scale 1-5

13. "나만 보기", "리더와 나", "소그룹 전체", "익명으로" 중 가장 안심되는 선택지는 무엇이었나요?  
    - Multiple choice
    - 나만 보기
    - 리더와 나
    - 소그룹 전체
    - 익명으로
    - 아직 잘 모르겠다

14. "기도했어요" 버튼은 부담 없이 누를 수 있었나요?  
    - Linear scale 1-5

15. 기도제목을 남길 때 불안하거나 헷갈린 점이 있었나요?  
    - Paragraph

## Section 5. Overall

16. 카톡에 기도제목을 쓰는 것과 비교했을 때 동행방의 좋은 점은 무엇이었나요?  
    - Paragraph

17. 카톡보다 불편한 점은 무엇이었나요?  
    - Paragraph

18. 다음 주에도 리더가 요청하면 동행방을 사용할 의향이 있나요?  
    - Multiple choice
    - 예
    - 상황에 따라
    - 아니오

19. 꼭 개선되면 좋겠는 한 가지는 무엇인가요?  
    - Paragraph

20. 마지막으로 남기고 싶은 의견이 있다면 적어주세요.  
    - Paragraph

## Success Signals To Watch

- 4번, 5번 평균 4점 이상: 첫 사용 진입이 충분히 쉽다.
- 7번에서 "예" 또는 "거의 가능했다"가 60% 이상: 20초 체크인 목표에 가깝다.
- 12번 평균 4점 이상: 공개 범위가 베타에 충분히 명확하다.
- 9번에서 "많이 그랬다"가 반복되면 UX 문구를 즉시 수정한다.
- 18번에서 "예" 또는 "상황에 따라"가 다수면 그룹 반복 사용 가능성이 있다.
